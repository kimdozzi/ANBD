# [쇼핑몰 서비스]

프로젝트 소개
- 스프링 강의를 들으면서 따라 해보고 기능 추가 구현 및 최적화를 목적으로 한다.
- 스프링부트와 JPA를 활용한 자바 웹 애플리케이션 개발이 주된 목적이다.
- 쇼핑몰 서비스를 개발하고, 더미 데이터를 넣어 하드웨어의 과부하를 캡쳐한다.
- 과부하를 줄이기 위한 학습을 진행하며 최적화를 한다.
